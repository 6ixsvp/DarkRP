Key Features

    /afk Command: Players type /afk in chat to enter AFK mode.
    AFK Job Reversion: Automatically switches the player to a pre-defined AFK job (TEAM_CITIZEN or custom).
    Lock Movement: Prevents AFK players from moving or interacting with the game.
    AFK Reward System: Optional in-game rewards for being AFK (e.g., earning money).
    Easy Customization: All editable settings are at the top of the script.

How to Use

    Copy and save the script to darkrpmodification/lua/darkrp_modules/afk_system.lua.
    Restart your server to enable the AFK system.
    Edit the settings (e.g., AFK job, rewards) to match your server's needs.
